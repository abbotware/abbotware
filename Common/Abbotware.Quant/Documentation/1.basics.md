# Functions

Even Function Definition:
$$
f(-x) = f(x), \forall x \in \mathbb{R}
$$
Even Function Properties:
$$
\int_{-a}^{a} f(x)dx = \int_{-a}^{0} f(x)dx + \int_{0}^{a} f(x)dx = 2\int_{0}^{a} f(x)dx
$$
Odd Function Definition:
$$
f(-x) = -f(x), \forall x \in \mathbb{R}
$$
Odd Function Properties:
$$
\int_{-a}^{a} f(x)dx = 0
$$
